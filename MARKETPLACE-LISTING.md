# Marketplace listing copy

Copy for places where fixed-price technical micro-services are welcome. Do not post this in random GitHub issues.

## Title

I will fix a small GitHub issue, docs bug, example, or test with a clean PR

## Short description

I handle tiny, bounded public GitHub fixes: documentation mistakes, broken links, README/install notes, examples, regression tests, and very small bug fixes. Send a public repo link plus expected/current behavior. If it is small and safe, I will deliver a focused PR or patch file with a short verification note.

## Packages

### Starter — $10

Docs, links, README/install notes, or tiny examples.

Delivery: focused PR or patch file + verification note.

### Standard — $25

Tiny test improvement, regression test, or very small public bug fix with clear verification.

Delivery: focused PR or patch file + verification note.

### Custom — $50+

Only after review. Must be bounded, public, low-risk, and externally verifiable.

## Buyer requirements

Please send:

1. Public GitHub repo URL.
2. Issue URL or exact problem statement.
3. Expected behavior.
4. Current behavior.
5. Acceptance criteria.
6. Test command or verification path, if known.
7. Confirmation that external PRs are accepted, or that patch-file delivery is OK.

## Exclusions

No private repositories, credentials, production access, paid tools, private APIs, customer data, security exploit work, scraping abuse, spam/outreach, auth/payment/wallet logic, KYC/tax/bank/identity work, mobile/hardware-only tasks, vague refactors, or broad architecture changes.

## Payment note

Payment accepted in USDC on Base, Arbitrum, Linea, or Ethereum after scope/price/ETA confirmation. Buyer confirms network before sending and covers network fees.
